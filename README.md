## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The app will be available at `http://localhost:3000`

## API Configuration

The app connects to: `https://api.similix.in`

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **UI**: Tailwind CSS + Custom Components
- **API**: Similix API (symptom search endpoint)
- **Icons**: Lucide React
- **Language**: TypeScript

## Usage

1. Enter symptoms in the search box (e.g., "headache with nausea")
2. Click Search or press Enter
3. View ranked remedy results with confidence scores
4. See matching symptoms for each remedy
